# FGI 결과 - 5단계: 세부 기능 탐색

## A. 메뉴 구성

### 배송 횟수 선호도

| 참가자 | 선호 횟수 | 이유 |
|--------|-----------|------|
| 윤서 | 주 2회 | 1인 가구, 냉장고 용량 제약, 출근일 수령 어려움 |
| 도형 | 주 1-2회 | 평일 야근/회식으로 활용도 낮음, 주말 위주 사용 |
| 지연 | 주 3회 | 4인 가족, 재택근무로 집밥 빈도 높음 |
| 석원 | 주 2-3회 | 평일 부담스럽지만 주말 위주로 활용 |
| 신철 | 주 2-3회 | 적당한 빈도, 너무 자주 오면 수령 부담 |

### 메뉴 선택 방식

**직접 선택 선호 이유:**
- **윤서**: 개인 취향 반영 (매운 음식, 해산물), 싫어하는 메뉴 배제
- **지연**: 알레르기 정보 정확한 확인 필요
- **신철**: 룸메이트와 취향 차이 고려

**AI 큐레이션의 조건부 수용:**
- **지연**: 알레르기 정보 정확 반영 시 가능
- 카테고리별 추천 시스템 (아이 친화적, 알레르기 프리 등) 선호

**가족 단위 고려사항:**
- **도형**: 부부 공통 취향 (고기, 파스타), 2인분 넉넉한 양
- **석원**: 4세 아이가 먹을 수 있는 자극적이지 않은 메뉴

## B. 배송 및 포장

### 배송 시간 선호도

**새벽 배송 선호:**
- **지연**: 즉시 냉장보관 가능, 신선도 유지, 당일 조리 가능

**저녁 배송 선호:**
- **윤서**: 출근일 수령 가능, 퇴근 후 즉시 조리

**유연한 배송 선호:**
- **신철**: 연구실 스케줄에 맞춘 배송 시간 조정 필요

### 포장재 환경 문제

**공통 우려사항:**
- 과도한 개별 포장 (플라스틱 용기, 비닐팩, 아이스팩)
- 분리배출 번거로움
- 환경 의식과 편의성 사이의 딜레마

**개선 요구사항:**
- **윤서**: 업체의 친환경 포장재 사용, 포장 최소화
- **신철**: 재활용 용기 + 회수 시스템, 가격 절약을 통한 환경 부담 상쇄
- **지연**: 포장재 적은 브랜드 선호도 증가

**현실적 접근:**
- 환경 중요하지만 편의성 완전 포기는 어려움
- 가격과 환경성의 균형점 필요

## C. 부가 서비스

### 조리 영상의 필요성

**높은 필요성:**
- **석원**: 요리 초보자에게 필수적, 실패 방지
- **지연**: 새로운 메뉴 도전 시 유용, 아이들 교육용

**보통 수준:**
- **도형**: 필요하지만 너무 복잡하면 오히려 부담

### AI 도우미 서비스

**긍정적 반응:**
- **석원**: "이거 어떻게 하지?" 실시간 질문 가능
- **지연**: 개인정보(알레르기) 기반 맞춤 안내

**단순함 선호:**
- **도형**: 기능은 좋지만 사용의 단순함이 중요

### 남은 재료 활용 레시피

**세그먼트별 차이:**
- **초보자 (석원)**: 추가 요리 도전 부담, 버릴 가능성 높음
- **숙련자 (지연)**: 관심 있지만 시간 제약으로 활용도 낮음
- **학생 (신철)**: 경제성 측면에서 관심 있을 수 있음

## 주요 인사이트

### 1. 개인화 요구의 중요성
- **알레르기 정보**: 지연님 사례처럼 필수적 요구사항
- **취향 반영**: 개인/가족 단위 선호도 학습 및 반영
- **라이프스타일**: 근무 패턴, 가족 구성에 따른 맞춤 서비스

### 2. 편의성 vs 환경성 딜레마
- 모든 참가자가 환경 문제 인식하지만 편의성 포기는 어려움
- **해결 방향**: 
  - 친환경 포장재 개발
  - 재활용 시스템 구축
  - 포장 최소화 기술

### 3. 세그먼트별 부가 서비스 니즈
- **초보자**: 실패 방지 중심 (영상, 실시간 도움)
- **숙련자**: 새로운 경험, 효율성 중심
- **가족**: 아이들과의 활동, 교육적 가치

### 4. 기술 서비스의 적정선
- 많은 기능보다는 **핵심 기능의 완성도**가 중요
- **직관적이고 간단한 인터페이스** 선호
- 복잡한 기능은 오히려 이탈 요인

## 설계 시사점

### 배송 옵션
- **유연한 배송 시간 선택제** 도입
- 개인 라이프스타일에 맞는 **맞춤 배송 스케줄**

### 메뉴 시스템
- **하이브리드 방식**: 직접 선택 + AI 추천
- **필터링 시스템**: 알레르기, 취향, 가족 구성 반영

### 포장 혁신
- **단계적 친환경화**: 포장재 줄이기 → 친환경 소재 → 회수 시스템
- **투명한 환경 정보 제공**: 탄소 발자국, 재활용률 등

### 부가 서비스 전략
- **레벨별 서비스**: 초보자 → 중급자 → 고급자
- **필수 vs 선택**: 핵심 기능과 부가 기능 구분
- **개인화 학습**: 사용 패턴 학습을 통한 맞춤 서비스 제공
